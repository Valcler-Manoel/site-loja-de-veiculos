> ⚠️ Este projeto possui caráter de protótipo

 **Propósito:** O projeto foi desenvolvido originalmente como parte prática para a disciplina de Fundamentos da Programação II, no Instituto Federal do Ceará em 2021.
 
Este foi um dos meus primeiros projetos de desenvolvimento web.  O objetivo principal era aplicar os fundamentos de desenvolvimento front-end e aprender boas práticas de estruturação de páginas e estilização. O site simula uma loja de veículos com uma proposta divertida, pois além de carros modernos e tradicionais, a interface conta com um catálogo de veículos raros vindos direto da cultura pop.

## Tecnologias Utilizadas

* **HTML5:** Utilizado para a estruturação semântica e organização de todos os blocos de conteúdo da página.
* **CSS3:** Responsável por toda a estilização visual, layout responsivo e design moderno da interface.
* **Font Awesome:** Biblioteca de ícones integrada ao CSS para enriquecer visualmente os menus, cards e rodapé.
* **JavaScript:** Utilizado de forma nativa para manipular o DOM, controlando a abertura do menu responsivo e o modal de login.
* **Swiper.js:** Biblioteca externa em JavaScript aplicada para criar os carrosséis de slides interativos das seções de veículos e avaliações.
